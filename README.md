# certificate-automation
🎓 Automated Certificate Generator

A one-click Excel VBA automation that generates personalized PDF certificates using a Microsoft Word template.

The application reads participant details from an Excel sheet, replaces placeholders in the Word template, exports each certificate as a PDF, and saves all generated certificates to a user-selected folder.

Perfect for workshops, training programs, schools, and events.

✨ Features
1. Read participant data from Excel
2. Automatically replace template placeholders
3. Generate one certificate per participant
4. Export certificates as PDF
5. Choose the output folder at runtime
6. One-click automation using VBA
   
🛠 Tech Stack
1. Microsoft Excel VBA
2. Microsoft Word Automation (COM)
3. PDF Export

🚀 How to Use
1. Open the Excel workbook.
2. Enter participant details.
3. Click Generate Certificates.
4. Select the Word template.
5. Choose the output folder.
6. All certificates are generated automatically.
   
⭐ Output
1. Personalized PDF certificates
2. Automatic placeholder replacement
3. Batch generation for multiple participants
